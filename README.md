# Number System Converter in C++

This project is a C++ program that converts numbers between different number systems.

Supported Number Systems:
- Binary
- Decimal
- Octal
- Hexadecimal



### Features
- Convert Binary to Decimal
- Convert Decimal to Binary
- Convert Octal numbers
- Convert Hexadecimal numbers

### Technologies Used
- C++
- Object Oriented Programming
- Arrays
- ASCII Logic
How it works:
1. Convert the input number to Decimal
2. Convert Decimal to the target base

Concepts Used:
- C++ Classes and Objects
- Arrays
- String Handling
- ASCII Logic
- Base Conversion Algorithm
