# Number System Converter in C++

This project is a C++ program that converts numbers between different number systems.

Supported Number Systems:
- Binary
- Decimal
- Octal
- Hexadecimal

How it works:
1. Convert the input number to Decimal
2. Convert Decimal to the target base

Concepts Used:
- C++ Classes and Objects
- Arrays
- String Handling
- ASCII Logic
- Base Conversion Algorithm
